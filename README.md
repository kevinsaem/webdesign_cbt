# 웹디자인개발기능사 문제풀이 사이트

> 학생들이 **변형된 문제**를 반복해서 풀어보고, 어려운 개념을 **초등학생도 이해할 수 있도록 쉬운 비유**로 학습할 수 있는 온라인 사이트.
> 서버·DB 없이 **HTML 파일만 열면 동작**하며, 모바일에서도 잘 보이는 반응형 디자인.

## 빠른 시작

```bash
# 그냥 index.html을 더블클릭해서 열어도 됩니다.
# 또는 로컬 서버로 실행:
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

GitHub Pages로 배포하면 별도 설정 없이 모바일에서도 바로 접속 가능합니다.

## 주요 기능

- 📝 **연습 모드** — 한 문제씩 채점하며 해설·쉬운 설명을 바로 확인
- ⏱️ **시험 모드** — 60문항(또는 가능한 만큼) 한꺼번에 풀고 합격선(60%) 채점
- 🔁 **오답 노트** — 틀린 문제만 골라 반복 학습 (브라우저에 자동 저장)
- 📚 **카테고리별 학습** — 색채/디자인일반 · HTML/CSS · JavaScript · UX/UI · 웹표준
- 🧒 **쉬운 개념 설명** — 동일 개념의 변형 문제 + 초등학생도 알 만한 비유
- 📱 **모바일 반응형** — 480px 이하 화면 최적화

## 파일 구조

```
index.html       — 메인 화면 (한 페이지에 모든 화면 포함)
styles.css       — 반응형 스타일
app.js           — 로직 (모드, 채점, 오답 저장, 화면 전환)
questions.js     — 문제은행 (원본 20문항 + 변형 25문항 = 45문항)
concepts.js      — 개념 사전 (20개 핵심 개념의 쉬운 비유)
자료실/          — 원본 정리 노트
```

## 문제 데이터 늘리기

`questions.js`에 객체를 추가하면 됩니다.
- 같은 개념의 변형 문제는 같은 `conceptId`를 사용 → 자동으로 묶임
- 새 개념을 추가하면 `concepts.js`의 `CONCEPTS`와 `questions.js`의 `CONCEPTS_INDEX`에도 추가
