# 웹디자인개발기능사 필기 — 정리 & 문제은행

> 온라인 문제풀이 사이트 제작용 참고 문서. 시험 개요, 출제기준, 학습 전략, 그리고 사이트에 바로 넣을 수 있는 문제 데이터(JSON 포함)를 담았습니다.

---

## 1. 시험 개요

| 항목 | 내용 |
|------|------|
| 자격 명칭 | 2025년부터 **웹디자인기능사 → 웹디자인개발기능사**로 변경 (영문명 Craftsman Web Design Development) |
| 시행 | 한국산업인력공단 (큐넷 Q-Net) |
| 응시 자격 | 제한 없음 (학력·경력·나이 무관) |
| 필기 방식 | 객관식 4지선다, **60문항** |
| 합격 기준 | **60문항 중 36문항 이상(60%)**, **과락 없음** |
| 평균 합격률 | 약 70~80% |
| 필기 면제 | 필기 합격 시 2년간 면제 |

### CBT 전환과 기출 비공개
- 2016년 10월부터 필기시험이 **CBT(Computer-Based Testing)** 방식으로 전환되면서 실제 시험문제가 **공개되지 않습니다.**
- 따라서 시중에 도는 자료는 대부분 **응시자 기억으로 복원한 문제** 또는 **출제경향 반영 예상문제**입니다.

---

## 2. 출제기준 (2025년 개편)

2025년부터 NCS 기준으로 출제 범위가 크게 개편되었습니다.

- **이전(~2024):** 디자인 일반 / 인터넷 일반 / 웹그래픽디자인 (3과목)
- **개편(2025~2027):** **웹디자인구현 / 웹페이지제작** (통합)

### 주요 출제 영역
- 프로토타입 기초데이터 수집 및 스케치 (기초데이터·레퍼런스)
- 프로토타입 제작 및 사용성 테스트
- 디자인 구성요소 설계·제작
- 구현 및 응용 (HTML / CSS / JavaScript)
- 조색과 배색 (색채)
- 프로젝트 완료 자료정리
- **UX/UI 설계, 와이어프레임, 스토리보드 작성** ← 2025 신설·강화 영역

> 공식 출제기준 문서는 큐넷 자료실 「웹디자인개발기능사 출제기준(2025.1.1~2027.12.31)」에서 다운로드.

---

## 3. 학습 전략

1. **출제기준 문서를 출발선으로** — NCS 기반 세부항목을 먼저 훑어 신설 단원(UX/UI, 프로토타입, 사용성 테스트) 파악.
2. **구 기출은 겹치는 단원만** — HTML·CSS·JavaScript 기초, 색채(조색·배색), 웹 그래픽, 인터넷 일반은 여전히 유효. 사라진 단원(Flash 등)은 건너뛰기.
3. **신설 영역은 별도 보강** — UX/UI·프로토타입·사용성 테스트는 옛 기출에 거의 없음. 용어 정의 위주(와이어프레임 vs 스토리보드 vs 프로토타입)로 암기.

### 키워드 검색 팁
자료를 찾을 때 **'웹디자인기능사'와 '웹디자인개발기능사'** 두 키워드를 모두 사용하면 폭넓게 검색됨.

## 5. 문제은행 (20문항)

영역 태그: `색채/디자인일반` · `HTML/CSS` · `JavaScript` · `UX/UI` · `웹표준`

<details>
<summary><b>문항 펼쳐보기 (정답·해설 포함)</b></summary>

**Q1. [색채/디자인일반]** 빛의 3원색(RGB)을 모두 합쳤을 때 나타나는 색은? (가산혼합)
① 검정 ② 흰색 ③ 회색 ④ 노랑
- **정답: ②** RGB는 더할수록 밝아지는 가산혼합으로, 세 색을 모두 합치면 백색이 된다.

**Q2. [색채/디자인일반]** 색의 3속성에 해당하지 않는 것은?
① 색상(Hue) ② 명도(Value) ③ 채도(Chroma) ④ 보색
- **정답: ④** 색의 3속성은 색상·명도·채도이며, 보색은 색상환에서 마주보는 색 관계다.

**Q3. [색채/디자인일반]** 먼셀 표색계에서 색을 표기하는 순서로 옳은 것은?
① 명도 채도 색상 ② 색상 명도/채도 ③ 채도 색상 명도 ④ 색상 채도/명도
- **정답: ②** 먼셀 표기는 'H V/C'(색상 명도/채도) 순. 예) 5R 4/14.

**Q4. [색채/디자인일반]** 두 보색을 인접시켰을 때 서로의 채도가 더 선명해 보이는 대비는?
① 명도대비 ② 계시대비 ③ 보색대비 ④ 연변대비
- **정답: ③** 보색을 나란히 두면 서로의 채도가 강조되는 보색대비가 나타난다.

**Q5. [색채/디자인일반]** 게슈탈트(Gestalt) 시지각 원리에 해당하지 않는 것은?
① 근접성 ② 유사성 ③ 폐쇄성 ④ 황금비
- **정답: ④** 게슈탈트 원리는 근접성·유사성·연속성·폐쇄성 등. 황금비는 비례 개념.

**Q6. [색채/디자인일반]** 형태를 이루는 가장 기본적인 조형요소가 아닌 것은?
① 점 ② 선 ③ 면 ④ 리듬
- **정답: ④** 점·선·면은 조형의 기본 요소, 리듬은 구성 원리.

**Q7. [색채/디자인일반]** 황금비율의 근사값으로 옳은 것은?
① 1 : 1.414 ② 1 : 1.618 ③ 1 : 2 ④ 1 : 1.732
- **정답: ②** 황금비는 약 1 : 1.618. 1:1.414는 금속비(√2).

**Q8. [HTML/CSS]** HTML5에서 그 자체로 완결되는 독립 콘텐츠를 정의하는 시맨틱 태그는?
① `<div>` ② `<span>` ③ `<article>` ④ `<b>`
- **정답: ③** `<article>`은 독립적으로 배포·재사용 가능한 콘텐츠. div·span은 의미 없는 컨테이너.

**Q9. [HTML/CSS]** 이미지를 표시할 수 없을 때 대체 텍스트를 제공해 웹 접근성을 높이는 `<img>` 속성은?
① src ② alt ③ title ④ href
- **정답: ②** alt 속성은 이미지 대체 텍스트로 스크린리더·접근성에 필수.

**Q10. [HTML/CSS]** HTML5 문서임을 선언하기 위해 문서 최상단에 작성하는 것은?
① `<meta>` ② `<!DOCTYPE html>` ③ `<head>` ④ `<title>`
- **정답: ②** `<!DOCTYPE html>`은 문서 형식 선언으로 맨 위에 작성.

**Q11. [HTML/CSS]** CSS 박스 모델에서 콘텐츠와 테두리(border) 사이의 안쪽 여백은?
① margin ② padding ③ outline ④ gap
- **정답: ②** 안쪽 여백은 padding, 테두리 바깥 여백은 margin.

**Q12. [HTML/CSS]** CSS 선택자 중 명시도(우선순위)가 가장 높은 것은? (인라인 스타일 제외)
① 태그 선택자 ② 클래스 선택자 ③ id 선택자 ④ 전체 선택자(*)
- **정답: ③** 명시도는 id > class > 태그 > 전체 순. 인라인 스타일은 그보다 높다.

**Q13. [HTML/CSS]** 요소를 화면에서 숨기면서 차지하던 공간도 없애는 CSS 선언은?
① visibility: hidden ② display: none ③ opacity: 0 ④ position: absolute
- **정답: ②** display:none은 공간까지 제거. visibility:hidden·opacity:0은 공간을 차지한다.

**Q14. [HTML/CSS]** 문서 흐름에서 벗어나 기준이 되는 부모 요소를 기준으로 배치되는 position 값은?
① static ② relative ③ absolute ④ sticky
- **정답: ③** absolute는 일반 흐름에서 빠져나와, position이 지정된 가장 가까운 조상 기준으로 배치.

**Q15. [JavaScript]** 블록 스코프를 가지며 재할당이 불가능한 변수 선언 키워드는?
① var ② let ③ const ④ new
- **정답: ③** const는 블록 스코프·재할당 불가. let은 재할당 가능, var는 함수 스코프.

**Q16. [JavaScript]** 사용자의 클릭·입력 등 동작에 웹 페이지가 반응하도록 하는 것을 무엇이라 하는가?
① 이벤트 ② 메서드 ③ 속성 ④ 주석
- **정답: ①** 클릭·입력·로드 등 발생하는 동작을 이벤트라 하며, 이벤트 처리기로 반응 구현.

**Q17. [UX/UI]** 화면 구성과 콘텐츠 배치를 선·도형으로 단순하게 표현한 설계 도면은?
① 스토리보드 ② 와이어프레임 ③ 프로토타입 ④ 무드보드
- **정답: ②** 와이어프레임은 레이아웃 골격을 단순한 선과 박스로 나타낸 설계도.

**Q18. [UX/UI]** 실제 동작에 가깝게 인터랙션을 구현해 사용성 테스트에 활용하는 산출물은?
① 사이트맵 ② 와이어프레임 ③ 프로토타입 ④ 스타일가이드
- **정답: ③** 프로토타입은 동작·전환을 구현해 실제 사용 흐름을 검증(사용성 테스트)하는 데 사용.

**Q19. [UX/UI]** 사용자가 제품을 사용하며 느끼는 전반적인 경험을 의미하는 용어는?
① UI ② UX ③ GUI ④ CLI
- **정답: ②** UX(User Experience)는 사용 전반의 경험, UI는 화면·조작 인터페이스.

**Q20. [웹표준]** HTML·CSS 등 웹 표준과 웹 접근성 지침을 제정하는 국제 기구는?
① ISO ② W3C ③ IEEE ④ ICANN
- **정답: ②** W3C(World Wide Web Consortium)가 웹 표준과 WCAG 접근성 지침을 제정.

</details>
